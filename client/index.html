HelloTodo

This simple todo list backend and web app were created using StrongLoop and Bluemix Mobile Services. Follow the onscreen guided experience to get started.

Getting Started

Add a few items to the list on the right.

Adding a new item

Well done. Two API calls were required to add that Todo:

POST - Create and add the new item
/api/Items
 GET - Retrieve the full list
/api/Items

Next, try editing an item.

Accessing an existing item

Good choice. One API call was required to retrieve the Todo information:

GET - Single item by ID.
/api/Items/:id

Next, hit save to store your changes.

Updating an item

Success. Two API calls were required to store your changes:

PUT - update item
/api/Items/:id
 GET - retrieve the full list 
/api/Items

Next, mark an item as completed using the checkbox.

Updating an item

Progress! Similarly, two API calls were required to store your changes:

PUT - update item
/api/Items/:id
 GET - retrieve the full list
/api/Items

Now, try to delete an item.

Deleting an item

An API call was made in an attempt to delete the todo item

DELETE - remove item
/api/Items/:id

However request failed since DELETE endpoint is protected by the Mobile Client Access service and can only be accessed by mobile applications instrumented with Mobile Client Access SDK.
